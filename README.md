# Airline Reservation System

## Project Overview

Airline Reservation System is a full-stack web application designed to simplify flight ticket booking and passenger management. The system allows users to register, securely log in, and book airline tickets by providing travel and passenger details. Booking information is stored and managed using MongoDB.

---

## Features

### User Authentication
- User registration with account creation.
- Secure login system using encrypted passwords.
- Password hashing implemented with bcrypt.

### Ticket Booking
- One-way and round-trip flight booking support.
- Passenger information collection and validation.
- Travel class selection and journey details management.

### Booking Management
- Stores booking records in MongoDB.
- Maintains passenger and travel information for future reference.

### Dynamic Web Interface
- Interactive user interface built using EJS templates.
- Responsive pages for registration, login, booking, and information sections.

## Workflow

1. User registers an account.
2. Password is securely hashed before storage.
3. User logs into the system.
4. User enters travel and passenger details.
5. Flight booking information is validated.
6. Booking data is stored in MongoDB.
7. Confirmation is generated after successful booking.
